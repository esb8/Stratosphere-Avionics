# Stratosphere
Stratosphere is a highly **versatile** avionics stack designed for a variety of **Remote-Controlled** (RC) vehicles such as planes, model rockets, and drones. Stratosphere is designed for *verstaility*, allowing for usage as a standalone stack and alongside off the shelf modules (VTX, GPS, etc), servos and ESCs. 

## Features
- **Flight Computer**
  - 6-axis IMU (ICM-42605)
  - High Precision Barometer (BMP388)
  - 915MHz LoRa Transciever (ELRS compatible)
  - 4 UART Outputs
  - 8 PWM capable pins
  - 4 PWM/Motor control pins (4-in-1 ESC compatible) 
  - STM32F405
  - USB-C programming
- **Power Management Unit**
  - 2-4S LiPo (7.4V-14.8V)
  - 3.3V 3A (3+ outputs)
  - 5V 3A (3+ outputs)
  - Compact + Affordable
  - XT60
